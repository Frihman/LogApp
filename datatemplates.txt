## Logs
Required features
-Id
-Content
-Date added
-Favourite (true/false)
-Mood (Attached emoji, img code?, for now only "sad" and "happy")
-Author (Id)

{
    "id" : "",
    "content" : "",
    "date" : "",
    "favorite" : "",
    "mood" : "",
    "authorId" : ""
}
